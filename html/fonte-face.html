Como usar @font-face

Faça algo incrível com a web Acesse: https://www.google.com/fonts# e procure a fonte que mais lhe agradar, em seguida clique no canto inferior direito no link com o nome "use". Na página que abre siga os passos:

  • Escolha o tipo de fonte, no caso a que você clicou.
  • Escolha o conjunto de caracteres, em geral o que já vem marcado será suficiente.
  • Copie o link do estilo para o cabeçalho do seu código html.
  • Dentro do seu css coloque as formatações com base na fonte escolhida. tipo font-family: 'Abril Fatface', cursive;

Outro exexpcional recurso que eu recomendo é o script webfontloader, com esse script você obtém maior controle sobre o recurso @font-face, podendo usar múltimplos repositórios de fontes.

A fonte do título é "Abril Fatface" e a fonte do corpo do texto é a fonte "Cardo", há outras possibilidades, veja o código fonte postado no github.
